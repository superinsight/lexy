# TOOLS.md - Local Notes

This file is for specifics unique to this attorney's setup and workflow.

## What Goes Here

Things like:

- Document storage paths and naming conventions
- Case management references
- Preferred citation formats
- Template locations
- Workflow-specific notes

## Examples

```markdown
### Documents

- Pleadings → ~/Documents/Cases/{client}/{matter}/pleadings/
- Research memos → ~/Documents/Cases/{client}/{matter}/research/

### Citation Style

- Default: Bluebook
- State filings: California Style Manual

### Templates

- Motion template → ~/Documents/Templates/motion-template.docx
- Letter template → ~/Documents/Templates/letter-template.docx
```

## Why Separate?

Your setup is yours. Keeping tool notes separate from personality (SOUL.md) and workspace rules (AGENTS.md) means each file stays focused and easy to maintain.
